Angular-filter-panel

A filter panel for any data source, made with AngularJS

Download .zip Download .tar.gz View on GitHub

Introduction

Requirements

AngularJS, obviously. Although really the magic of the filter panel is possible thanks to the amazing Underscore.js. So don't forget to include it in your project! Also, the template of the filter panel is styled using Bootstrap classes, but if you don't want to include Bootstrap in your project it's ok. Simply keep in mind that, once rendered, the filter panel will probably look weird and quite ugly if you did not include Bootstrap.

Instructions

Obviously this array of objects would very probably be asynchronously populated by an AJAX request!

It's very easy to include a filter panel in your HTML page. The filter panel is restricted to element, and you need to provide it with two attributes only.

source

This is your data source. Don't worry, the filter panel won't tamper your data, it simply wants to play with it.

filter-props

These are the names of the properties in your data source objects that you want the data to be filtered by. Of course you could specify them all, but it makes sense to filter your data only by those properties which hold non-unique values.

<filter-panel source="dataSource" filter-props="Code,Sport,Country,Enabled"></filter-panel>

Whenever your data source changes, the filter panel will process it, grouping the data by the specified properties and generating the filter groups. The available filter values will be displayed on the filter panel.

Once you select some filter values and click APPLY, the filter panel performs its magic.

Id Client Code Sport Enabled Country
{{item.Id}} {{item.Client}} {{item.Code}} {{item.Sport}} {{item.Country}}
